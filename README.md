## 😫 你是不是也这样？

做了一上午的装配体，零件越来越多——

然后屏幕一闪：**"您的系统内存不足"**

你打开任务管理器，看着一长串看不懂的进程名，不知道哪个能关、哪个关了会出事……

## ✨ 内存急救包能帮你

| 功能 | 说明 |
|------|------|
| 🔍 智能扫描 | 自动列出所有占内存的进程，按大小排序 |
| 🏷️ 分类标签 | 🌐浏览器 💬通讯 🎵媒体 ☁️云盘 🎮游戏... 一目了然 |
| 🚦 风险评分 | 🟢安全 🟡注意 🔴危险 — 不用猜，直接告诉你能不能关 |
| 💡 悬停说明 | 鼠标移上去看详情：这个进程是什么？关了会怎样？ |
| 🛡️ 自动保护 | SolidWorks 进程绝不误关，系统核心进程也不显示 |
| 🚀 一键释放 | 勾选 → 确认 → 3秒搞定 |

## 📥 下载安装

### 方式一：直接下载 .exe（推荐）

前往 [Releases 页面](https://github.com/releases) 下载最新的 `内存急救包.exe`

双击即可运行，**无需安装**，不需要 Python 环境。

### 方式二：从源码运行

```bash
git clone https://github.com/xxx/memory-cleaner.git
cd memory-cleaner
pip install -r requirements.txt
python memory_cleaner.py
```

## 📖 使用说明

1. **双击打开** 内存急救包.exe
2. 自动扫描当前运行的进程
3. 🟢安全进程已自动勾选，可根据需要增减
4. 点击 **「一键释放选中进程」**
5. 确认后自动关闭并刷新

### 风险等级说明

| 等级 | 含义 | 默认勾选 | 示例 |
|------|------|---------|------|
| 🟢 安全 | 关闭无副作用 | ✅ 是 | Steam、百度网盘、网易云音乐 |
| 🟡 注意 | 关闭有影响，需确认 | ❌ 否 | Chrome（丢标签页）、微信（断消息） |
| 🔴 危险 | 不建议关闭，可能丢数据 | ❌ 否 | 未保存的 Excel、正在编辑的文档 |

### 保护名单

以下进程**不会出现在列表中**，绝对不会被误关：

- ✅ SolidWorks 全系列进程（sldworks.exe 等）
- ✅ Windows 系统核心（explorer.exe、svchost.exe 等）
- ✅ 安全软件（Windows Defender 等）
- ✅ 硬件驱动程序
